# ToDo App (Java)

A simple console-based ToDo application built in Java.  
It allows users to:

- View all tasks
- Add a task
- Edit a task
- Mark a task as completed or not completed
- Remove a task

## How to Run
1. Compile the Java file:
   javac Main.java

2. Run the program:
   java Main

## Example
Menu:
1. View all tasks
2. Add a task
3. Edit a task
4. Mark a task as completed
5. Remove a task
6. Exit the application
